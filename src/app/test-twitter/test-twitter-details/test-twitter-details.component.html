Twitter handle detection

Twitter handle detection is the process of accessing live tweets from the user. The social media platform to be used is Twitter . Depression is a medical illness that affects an individual negatively by changing the way one feels, think and act, but luckily, it's treatable. So, the problem is its detection which we can solve with the help of machine learning. Textual interactions of an individual in the form of tweets, status updates and posts also hold data with respect to the mental state of an individual.

How it Works

  • Text pre-processing: Text features are extracted and pre-processed.
  • Scraping: Live tweets data using tweepy undergoes scraping phase where scraping of text data is processed.
  • Twitter handle input: Live tweets are analysed in the scraping phase and the generated input data is fed into the system to handle the live twitter data.
  • Creation of a model: The model is created using five machine learning algorithms namely, SVM (Support Vector Machine), Decision Tree, Na√Øve Bayes, KNN (K-Nearest Neighbors) and Random Forest to handle the text data and hence trained which yields the result as either depressed or not.
